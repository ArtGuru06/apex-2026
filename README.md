# APEX 2026 — College Tech Fest Website

A premium, full-featured event management website for **APEX 2026** — a college tech fest with multiple club stalls, live leaderboards, and student check-in system.

## 🚀 Live Demo
[View Live →](https://Armaanjot18.github.io/apex-2026)

## ✨ Features

- **Student Portal** — Sign up once, get a unique Student ID, then check in to any club stall
- **Club Portal** — Each club gets a secure login to monitor their stall in real-time
- **Live Dashboard** — Clubs see visitor count, ratings, and student list updating live
- **Public Leaderboard** — Anyone can see which clubs have the most visitors & highest ratings
- **Star Rating System** — Students rate their experience (1–5 stars) with optional comments
- **Secure Auth** — Password hashing, session management, duplicate check-in prevention
- **Fully Responsive** — Works on desktop, tablet & mobile

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **Vanilla CSS** — Custom premium dark theme with glassmorphism & animations
- **Vanilla JS** — SPA routing, real-time updates, localStorage persistence
- **No frameworks** — Zero dependencies, works offline

## 📁 Project Structure

```
apex-2026/
├── index.html     # Main HTML (all pages/views)
├── styles.css     # Premium dark theme stylesheet
├── app.js         # Application logic & routing
├── data.js        # Data layer (localStorage DB)

```
